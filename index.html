pip install -U SaavnAPI

Quickstart

Get up and running in seconds. SaavnAPI uses Python's asyncio — use it inside any async context or with asyncio.run().

python
import asyncio
from SaavnAPI import SaavnClient

async def main():
    async with SaavnClient() as client:
        results = await client.search_songs("Kesariya")
        for song in results:
            print(song.title, "-", song.download_url)

asyncio.run(main())
💡SaavnAPI supports Python's async context manager (async with) to automatically manage the HTTP session lifecycle.

API Reference

All methods are coroutines and must be awaited inside an async function.

class SaavnClient

The main client class. Manages the underlying HTTP session.

python
from SaavnAPI import SaavnClient

# Recommended: context manager
async with SaavnClient() as client:
    ...

# Manual session
client = SaavnClient()
await client.start()
...
await client.close()

Examples

🤖 Telegram Music Bot

python
from SaavnAPI import SaavnClient

async def handle_song_request(query: str) -> str:
    async with SaavnClient() as client:
        songs = await client.search_songs(query, limit=1)
        if not songs:
            return "No songs found."
        song = songs[0]
        return f"🎵 {song.title}\n👤 {song.artist}\n💿 {song.album}\n🔗 {song.download_url}"

🎤 Lyrics Fetcher

python
import asyncio
from SaavnAPI import SaavnClient

async def get_lyrics(song_name: str):
    async with SaavnClient() as client:
        results = await client.search_songs(song_name, limit=1)
        if results:
            lyrics = await client.get_lyrics(results[0].id)
            print(lyrics)

asyncio.run(get_lyrics("Kal Ho Naa Ho"))
